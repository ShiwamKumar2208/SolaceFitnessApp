# Solace - Fitness App

![Preview](logo.png)

Solace is a minimal, offline-first fitness web app designed to turn
beginners into consistent, disciplined individuals over 90+ days.

No accounts. No clutter. No decision fatigue.

Just open the app and start.

https://solacefitnessapp.netlify.app/
 
## 🚀 Core Idea

Most fitness apps fail beginners because they: - overwhelm with
choices - require motivation - add unnecessary features - paid to use

Solace does the opposite.

> It removes thinking and forces consistency.
 
## ⚙️ Features

-   🧠 90+ day progressive workout system (auto-generated)
-   📉 Beginner → Intermediate progression
-   🔁 Infinite mode after day 90
-   📆 Daily structured workouts
-   🔥 Streak tracking
-   ⏱ Timer + rest system
-   ⏭ Skip + pause controls
-   📊 Exercise progress tracking
-   🖼 Exercise images + video guidance
 
## 📱 PWA (Progressive Web App)

-   📲 Installable on mobile
-   ⚡ Works offline
-   🚀 Fast loading via caching 
 
## 🧩 Tech Stack

-   Vanilla JavaScript
-   HTML + CSS
-   Service Workers (offline support)
-   LocalStorage (data persistence)
 
## 🎯 Philosophy

Solace is not a fitness tracker.

It is a **discipline system**.

No scrolling.\
No choosing workouts.\
No excuses.
 
## 🛠 Future Improvements
